Import File Requirements

Uploaded files must be:

  • A zipped ('.zip') file containing the elements of a valid ESRI shapefile, including files of type:
    • .dbf
    • .prj
    • .shp
    • .shx
    • NOTE: The above file types are the bare minimum. Your shapefile may come with more files. Please include them.
  • Files should be smaller than 10Mb.
    • If larger, please contact the site administrator for more options for including this data
  • Projected into EPSG:3857, aka "WGS 1984 Web Mercator (Auxillary Sphere)"
